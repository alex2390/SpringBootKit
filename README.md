- 集成 MongoDB
  - 支持创建、删除集合，自动生成下一自增 id
  - 创建唯一索引，删除索引，查询所有索引
  - 查询，模糊查询，查询集合，模糊查询集合，分页查询集合，模糊分页查询集合
  - 新增，批量新增
  - 更新
  - 删除
- 简单集成注解式 webSocket
- 存储服务器本地文件上传、下载
- 加密、解密算法：MD5,RSA
- 集成并封装通用 Mapper
- EasyExcel（导入、导出）
- Twitter 的分布式 id 生成算法
- 邮箱（支持群发，多附件）
- 定时器
- Redis 工具类（暂无集群）
- HttpUtil 工具类（RESTful API 形式）
- JSON 工具类（FastJson处理）
- Date,String,PhoneNumber,Math 工具类
- servlet,listener,filter,interceptor,runner,aspect
- 十大排序算法和23种设计模式
